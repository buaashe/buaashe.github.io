A precision-oriented tool for Over-approximations and Under-approximations of Reachable Sets

TOOLBOX DESCRIPTION

We present

OURS

, a precision-oriented MATLAB tool for computing Over- and Under-approximations of Reachable Sets for nonlinear analytic differential equations. Since the tool is written in MATLAB, the installation and use are platform independent. The following points summarize the main capabilities of the

OURS

:

  • 1. The current version of OURS allows rational functions, trigonometric, and even exponential functions with arbitrary nestings in the input.

  • 2. OURS can obtain over-and under-approximations of reachable set at any desired instant in the time interval from obtained explicit expressions without repeated computation.

  • 3. OURS can directly handle non-convex and even non-connected sets .

  • 4. OURS can deal with time-varying analytic systems with disturbances described by uncertain parameters.
